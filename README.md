# SAL Tools

A minimal VS Code extension for SAL.

## Features

- Syntax highlighting for `.sal` files.
- `SAL: Install from GitHub Release` downloads a release asset and sets `sal.toolchain.binPath`.
- `Hello World: Say Hello` remains as the tiny starter command.

## Install SAL

Run `SAL: Install from GitHub Release` from the command palette.

By default, the command installs `karthiknukala/sal` from the `nightly` release tag. Set `sal.install.releaseTag` to an empty string to use the latest release endpoint instead.

## Run the Extension

1. Open this folder in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. Open a `.sal` file or run a command from the command palette.
